# Audio Browser

A lightweight utility module for Foundry Virtual Tabletop that adds a convenient **Audio Browser** button to the **Ambient Sounds** controls.

The Audio Browser provides quick access to Foundry's built-in audio file manager, making it easier to browse, upload, and organise audio files without leaving the canvas.

---

## Features

- Adds an **Audio Browser** button to the Ambient Sounds controls.
- Opens Foundry's built-in audio file browser.
- Supports browsing existing audio files.
- Supports uploading new audio files.
- Allows creation and organisation of audio folders.
- Configurable default opening folder.
- GM-only access.

---

## Installation

Install the module using Foundry's standard module installation process and enable it within your world.

---

## Usage

### Opening the Audio Browser

1. Open a scene.
2. Select the **Ambient Sounds** controls from the left-hand toolbar.
3. Click the **Audio Browser** button.
4. The browser will open in your configured default audio folder.

---

### Uploading Audio Files

1. Open the Audio Browser.
2. Navigate to the desired folder.
3. Click the **Upload** button.
4. Select your audio file(s).
5. Once uploaded, the files will be available for use throughout Foundry.

---

### Setting a Default Audio Folder

1. Open **Game Settings**.
2. Select **Configure Settings**.
3. Open the **Module Settings** tab.
4. Locate **Audio Browser**.
5. Enter the folder path you would like the browser to open by default.
6. Save your changes.

---

## Compatibility

- Foundry Virtual Tabletop V13+

---

## Permissions

The Audio Browser is available to **Game Masters only**.
